# Radency Hometask 2

## Run Locally 

Clone the project

```bash
    git clone git@github.com:yegor-ts/task-2.git
```

Go to the project directory

```bash
    cd task-2
```

Install dependencies    

```bash
    npm install
```

Run application with auto reload

```bash
    npm start
```

## Lint

- Configuration for eslint rules in `.eslintrc`

- To run lint
```bash
    npm run lint
```
